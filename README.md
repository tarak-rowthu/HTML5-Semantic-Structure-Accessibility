# 📚 Tarak Naga Venkata Durgesh Rowthu Portfolio

---

## ✨ Introduction

A sleek, modern **multi‑page personal portfolio website** showcasing my work, skills, and contact information. Built with **HTML5, CSS3, JavaScript, React, and Vite**, the site delivers a fast, accessible, and responsive experience across all devices.

---

## 📖 About the Portfolio

This project is a personal showcase that highlights:

- Clean, semantic markup for SEO and accessibility.  
- Responsive layouts that adapt seamlessly from mobile to desktop.  
- Interactive navigation and a dynamic projects gallery.  
- A functional contact form ready for integration with any backend service.

---

## 🚀 Features

| Feature | Description |
|---|---|
| **Semantic HTML5 Structure** | Uses proper headings, landmarks, and ARIA attributes for optimal SEO and screen‑reader support. |
| **WCAG Accessibility** | Implements color contrast, focus management, keyboard navigation, and ARIA roles to meet WCAG AA standards. |
| **Responsive Design** | Fluid grid and CSS custom properties ensure the site looks great on all screen sizes. |
| **Multi‑Page Navigation** | Client‑side routing via React Router for instant page transitions. |
| **Projects Showcase** | Interactive cards with filters, tags, and modal previews. |
| **Contact Form** | Styled, validated form ready for integration with services like Formspree or Netlify Forms. |
| **SEO Optimization** | Meta tags, Open Graph protocol, and structured data for better search visibility. |

---

## 🛠️ Technologies Used

- **HTML5** – Semantic markup and structure.  
- **CSS3** – Flexbox, Grid, custom properties, and animations.  
- **JavaScript (ES6+)** – Core logic and interactivity.  
- **React** – Component‑based UI and state management.  
- **Vite** – Fast dev server and optimized build pipeline.

## ⚙️ Installation & Local Setup

1. **Clone the repository**

   ```bash
   git clone <GitHub Repository URL>
   cd portfolio
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run the development server**

   ```bash
   npm run dev
   ```

4. Open `http://localhost:5173` (or the port shown in the terminal) in your browser.
